JavaScript Continue Statement

Explanation:

The continue statement skips the current loop step.
JavaScript moves directly to the next repetition.

Example Code:

let i = 0;
let text = "";

while(i < 5)
{  // Increase number first
   i++;
// Skip number 3
   if(i == 3)
   {
      continue;
   }

   text += "<br>The number is " + i;;
}
